Comme il n’existe pas d’emplacement normalisé pour entreposer les fontes qui ne sont pas installées, la première chose à faire lorsque vous lancez Fontmatrix pour la première fois est d’importer les fichiers de fontes que vous souhaitez gérer à l’aide de Fontmatrix.

Comprendre et utiliser l'interface

Intro

In the main view, each font is drawn with its typeface. You can choose and customize the text to be drawn in this view for every font face. Custom text can include Unicode symbols.

You can preview typefaces of installed fonts listed in the Windows Fonts view, typefaces of fonts listed in the List view, or you can select any folder in the Folders view preview all typefaces located in the selected folder.

Gérer les fontes

Importer des fontes

Pour ce faire, deux moyens sont à votre disposition. Le premier consiste à sélectionner l’entrée Importer... du menu Fichier. Il vous sera alors demandé de choisir un répertoire, lequel sera exploré afin d’importer les fichiers de fonte qu’il contient --- en manière de facilité, Fontmatrix effectue une récursion dans les répertoires contenus dans celui que vous avez choisi. Le second vise à importer un choix de fichiers dans Fontmatrix et s’effectue par un glisser-déplacer.

L’interface

La fenêtre principale de Fontmatrix est séparée en deux. À gauche, par défaut, se trouvent les listes des fontes dont Fontmatrix a connaissance. L’une contenant les noms et l’autre des aperçus. L’autre partie présente un ensemble d’onglets contenant chacun un type d’information relatif à la fonte présentement sélectionnée. Informations regroupe les informations collectées dans les champs prévus à cet effet par le format de fichier de fonte, plus, pour les fontes OpenType, l’arborescence des fonctionnalités. Texte insensé est un aperçu paramétrable de la fonte « en action ». Dessins présente tous les glyphes référencés par la table des caractères Unicode, par morceaux. Et enfin Étiquettes vous permet d’associer des mots à vos fontes.

You can submit bugs at bugtracker (no need to register), or join the mailing-list – don’t worry about the "Undertype" name, as Fontmatrix primarily belongs to this bigger project, even if it’s now independent.
