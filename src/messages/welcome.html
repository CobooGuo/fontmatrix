##WELCOME_FONT##
Fontmatrix is a user-friendly font manager for linux. Regarding it's clear and nice user interface, it is very simple to learn and use.
Fontmatrix has some unique functionalities in the Open Source world : - Activate, deactivate fonts to reduce font lists and save ressources - Allow to tag font for a better classification and group (de)activation - make it easy to find particular glyph thanks to a glyph category display - Create a font book that can be printed to have a real overview of the fonts available - Display OpenType options - Can grab fonts stored in distant directories, for example when they are shared in an enterprise or a network.
We hope you'll enjoy using Fontmatrix.
