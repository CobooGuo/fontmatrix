What's new

The version you are looking at contains a number of improvements over the previous one, such as:

  • a rather improved user interface;
  • searching for fonts by selecting PANOSE features;
  • searching for fonts by using bitmap samples of glyphs;
  • a possibility to edit PANOSE metadata without saving changes to actual fonts;
  • glyphs comparison, with optional color fill, points and metrics display;
  • extracting subsetted fonts from PDF;
  • a new help system, with search and bookmarks;
